# Test Plan: BankPro Application

## 1. Overview

BankPro is a banking application that allows users to manage their accounts, make transfers and perform financial operations.

## 2. Test Objective

The goal of testing is to verify that the main application features work correctly and securely.

## 3. Scope of Testing

### In Scope:

- User registration
- User authorization
- Authentication
- Money transfers
- User profile
- Transaction history
- API responses


### Out of Scope:

- Server infrastructure testing
- Load testing
- Third-party payment systems testing

## 4. Types of Testing

- Functional Testing
- Regression Testing
- Smoke Testing
- Exploratory Testing
- API Testing
- Security Testing


## 5. Testing Tools

- Jira
- Postman
- Chrome DevTools
- GitHub
- SQL


## 6. Test Environment

Platform:
Android

Application:
BankPro


## 7. Risks

- Incorrect transaction processing
- Authentication issues
- Data synchronization problems
- Security vulnerabilities


## 8. Entry Criteria

Testing starts when:

- Application build is available
- Requirements are provided
- Test environment is ready


## 9. Exit Criteria

Testing is completed when:

- Critical bugs are fixed
- Main scenarios are verified
- Test documentation is completed
