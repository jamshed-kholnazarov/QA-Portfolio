## Test case ID

TC-001

## Title

Registration with valid user data

- User is on registration page
- User does not have an existing account

## Test steps

1. Open MarketShop application
2. Click "Registration"
3. Enter valid name
4. Enter valid email
5. Enter valid password
6. Confirm password
7. Click "Create account"

## Expected result

- User account is created successfully
- User is redirected to personal account
- User receives authentication token
- User data is saved
